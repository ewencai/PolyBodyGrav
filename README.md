# PolyBodyGrav

Gravity forward modeling for regular-shaped polyhedral bodies.

## Features

- Forward gravity modeling for rectangular prisms, spheres, cylinders
- Full tensor gravity gradiometry (FTG) components
- Interactive parameter adjustment
- Anomaly map visualization

## System Requirements

- Windows 10/11 (64-bit)
- MATLAB Runtime R2026a

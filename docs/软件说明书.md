## 1. 软件概要

**规则体重力正演系统** 是一款基于 MATLAB 的桌面应用，用于计算和可视化七种规则几何体（球体、长方体、各类圆柱体、椭球体、截锥体、多面体）产生的重力异常及其梯度张量（共 7 个分量）。支持单规则体正演、多规则体叠加正演、参数敏感性分析，以及网格数据多格式导出。

**技术特性：**

- 长方体与球体采用严格解析公式（7 分量全解析），其余形体采用解析+数值有限差分混合算法
- 球体、长方体重力梯度张量满足自由空间 Laplace 方程（Vxx+Vyy+Vzz=0）
- 三维可视化（Gouraud 光照渲染）+ 平面等值线图 + 敏感性曲线族
- 观测面支持常量平面与外部地形曲面文件（ASC/GRD/DAT/TXT/XLSX）
- 网格导出支持 ESRI ASCII Grid、Surfer 6 Grid、XYZ 列式三种 GIS 标准格式
- 参数持久化与 JSON 导入/导出，支持跨会话恢复

**系统环境：** MATLAB R2025a 或更新版本，无需额外工具箱。

**启动方式：** 运行 `Main_App.m` 或直接调用 `ui.RegularBodyApp()`。

## 4. 技术原理与公式

### 4.1 万有引力常数

所有计算使用 CODATA 2018 推荐值：

$$G = 6.67430 \times 10^{-11} \ \text{m}^3 \cdot \text{kg}^{-1} \cdot \text{s}^{-2}$$

### 4.2 球体 — 点质量解析公式

将球体等效为位于球心的点质量，密度 ρ (g/cm³) → ρ_si = ρ × 1000 (kg/m³)：

$$M = \frac{4}{3}\pi R^3 \rho_{si}$$

$$V_z = G M \frac{-dz}{r^3} \times 10^8 \quad (\mu\text{Gal})$$

$$V_{zz} = G M \frac{3 dz^2 - r^2}{r^5} \times 10^9 \quad (\text{E})$$

$$V_{xx} = G M \frac{3 dx^2 - r^2}{r^5} \times 10^9, \quad V_{yy} = G M \frac{3 dy^2 - r^2}{r^5} \times 10^9$$

$$V_{xy} = G M \frac{3 dx \cdot dy}{r^5} \times 10^9, \quad V_{xz} = G M \frac{3 dx \cdot (-dz)}{r^5} \times 10^9, \quad V_{yz} = G M \frac{3 dy \cdot (-dz)}{r^5} \times 10^9$$

其中 $dx = x - X_0$, $dy = y - Y_0$, $dz = z_{obs} - Z_0$, $r = \sqrt{dx^2 + dy^2 + dz^2}$。

### 4.3 长方体 — Nagy 棱柱公式

采用 Plouff (1976) 角点叠加法。设长方体 8 个角点分别为 $(x_i, y_j, z_k)$，角点重力核函数 GH 为：

$$GH(x,y,z; x_i,y_i,z_i) = (x_i-x)\ln[(y_i-y)+R] + (y_i-y)\ln[(x_i-x)+R] - (z_i-z)\arctan\frac{(x_i-x)(y_i-y)}{(z_i-z)R}$$

其中 $R = \sqrt{(x_i-x)^2 + (y_i-y)^2 + (z_i-z)^2}$。

$$V_z = -G\rho_{si} \sum_{i=1}^{2}\sum_{j=1}^{2}\sum_{k=1}^{2} (-1)^{i+j+k} \cdot GH(x,y,z; x_i, y_j, z_k)$$

**重力梯度张量**（解析，Plouff 符号约定）：

$$V_{xx} = G\rho_{si} \sum \pm \arctan\frac{yz}{xR}, \quad V_{yy} = G\rho_{si} \sum \pm \arctan\frac{xz}{yR}, \quad V_{zz} = G\rho_{si} \sum \pm \arctan\frac{xy}{zR}$$

$$V_{xy} = -G\rho_{si} \sum \pm \ln|z+R|, \quad V_{xz} = -G\rho_{si} \sum \pm \ln|y+R|, \quad V_{yz} = -G\rho_{si} \sum \pm \ln|x+R|$$

所有梯度均乘以 $10^9$（转换为 Eötvös 单位）。

### 4.4 垂直圆柱体

轴向项与几何衰减因子的乘积形式，Vz 解析计算。梯度分量通过有限差分获得（见 §4.10）。

### 4.5 水平圆柱体 — 有限线质量

设线密度 $\lambda = \pi R^2 \rho_{si}$（线质量，kg/m），半长 $L/2$：

$$V_z = 2G\lambda \cdot \frac{|Z-z|}{d^2} \cdot \frac{L/2}{\sqrt{d^2 + (L/2)^2}}$$

其中 $d$ 为观测点到圆柱轴线的水平距离。梯度分量通过有限差分获得。

### 4.6 倾斜圆柱体 — Blakely (1996) 线积分

基于 Blakely (1996, eq. 3.29) 的有限线质量解析积分。对任意空间方向的线段，将观测点投影到线段所在直线上，计算端点的 cos/sin 角度项后叠加。梯度分量通过有限差分获得。

### 4.7 椭球体 — 高斯-勒让德数值积分

采用 32 节点 Gauss-Legendre 积分对椭球体表面进行数值积分。观测点所在的椭球面参数 $\lambda$ 通过 Newton-Raphson 迭代（`solve_lambda_ell`）求得。坐标变换包含三次欧拉旋转（Z-Y-X）。梯度分量通过有限差分获得。

### 4.8 截锥体 — 薄盘积分

将截锥体沿 Z 方向离散为薄盘序列，半径线性变化：

$$R(z) = R_{top} + (R_{bottom} - R_{top}) \cdot \frac{z - z_{top}}{z_{bottom} - z_{top}}$$

采用梯形积分法则（端点半权），离散步长 $\Delta z = \max(1.0, |z_b-z_t|/200)$。梯度分量通过有限差分获得。

### 4.9 多面体 — Werner/Scheeres 公式

对每个面计算从观测点出发的立体角 $\Omega$：

$$g_z(\mathbf{r}) = -G\rho \sum_{faces} \mathbf{n}_z \cdot h \cdot \Omega(\mathbf{r}, face)$$

其中 $h$ 为观测点到面所在平面的有符号距离。梯度分量通过有限差分获得。

### 4.10 数值梯度 — 有限差分法

对除球体和长方体外的所有形体，Vz 为解析解，梯度张量通过三层有限差分获得：

1. 在三个 Z 高程（$z_{obs}$、$z_{obs}+dz$、$z_{obs}-dz$）分别计算 Vz
2. $V_{zz} = (V_z^{top} - V_z^{bottom}) / (2 \cdot dz) \times 10^9$
3. $V_{xz} = \partial(V_z) / \partial x \times 10^9$，$V_{yz} = \partial(V_z) / \partial y \times 10^9$（MATLAB `gradient` 函数）
4. 由自由空间 Laplace 方程约束：$V_{xx} = V_{yy} = -V_{zz} / 2$
5. $V_{xy} \approx 0$（三层 Z 采样无法分辨水平交叉梯度）

**有限差分步长：** $dz = 1.0$ m

### 4.11 单位换算

- **Vz（重力）：** SI 值 (m/s²) → μGal，乘以 $10^8$
- **梯度张量（Vzz, Vxx, Vyy, Vxy, Vxz, Vyz）：** SI 值 (1/s²) → Eötvös (E)，乘以 $10^9$

$$1 \ \mu\text{Gal} = 10^{-8} \ \text{m/s}^2, \quad 1 \ \text{E} = 10^{-9} \ \text{s}^{-2}$$

## 5. 重力分量与单位

| 分量 | 物理含义 | 显示单位 |
|---|---|---|
| Vz | 重力垂直分量 $g_z$ | μGal |
| Vzz | 垂直梯度 $\partial g_z / \partial z$ | E (Eötvös) |
| Vxx | 水平梯度 $\partial g_x / \partial x$ | E |
| Vyy | 水平梯度 $\partial g_y / \partial y$ | E |
| Vxy | 交叉梯度 $\partial g_x / \partial y$ | E |
| Vxz | 交叉梯度 $\partial g_x / \partial z$ | E |
| Vyz | 交叉梯度 $\partial g_y / \partial z$ | E |

所有 7 个分量均可通过"正演量"下拉菜单选择。

## 6. 操作步骤

### 6.1 基本正演计算

1. 启动软件：运行 `Main_App.m`
2. 点击顶部选项卡选择规则体类型（如"球体"）
3. 在左侧"坐标参数"区设置网格范围、间距和观测面高程
4. 在"规则体参数"区设置几何参数和密度
5. 选择正演分量（Vz / Vzz / ...）和色标方案
6. 点击 **计算**
7. 查看右侧：上半部分为平面等值线图 + 观测面地形，下半部分为三维规则体模型
8. 可使用 MATLAB 图形工具栏旋转、缩放三维视图

### 6.2 使用曲面观测面

1. 将"观测面类型"从"常量平面"切换为"曲面文件"
2. 点击显现的 **加载观测面** 按钮
3. 选择地形文件（支持 `.asc` / `.grd` / `.dat` / `.txt` / `.xlsx`）
4. 文件需为与网格匹配的 2D 高程矩阵
5. 加载后观测面地形图自动更新

> `testdata/` 目录下提供了 7 种示例地形（valley / mountain / slope_x / basin / complex / flat / random），每种均有 5 种格式。

### 6.3 导出正演结果

**导出列式数据：** 点击 **导出数据** → 选择 `.txt` 或 `.xlsx` → 每列为一个分量的所有格点值（展平为一维）。

**导出 GIS 网格：** 点击 **导出网格** → 选择格式（`.asc` / `.grd` / `.dat`） → 选择"正演值"或"基准面" → 保存为带地理坐标头的 2D 网格文件。

### 6.4 参数保存与加载

- **保存参数：** 点击 **保存参数** → 选择保存路径 → 当前面板所有参数写入 UTF-8 JSON 文件
- **加载参数：** 点击 **加载参数** → 选择 JSON 文件 → 参数自动恢复到面板控件
- **自动持久化：** 退出程序时参数自动保存至 `app_settings.mat`，下次启动自动恢复

### 6.5 参数敏感性分析

1. 在规则体正演选项卡中设置好基准参数
2. 切换到 **参数敏感性分析** 子选项卡
3. 选择 X 轴变量和变化范围（Min / Max）
4. 设置步长数（默认 20）
5. 选择曲线族变量（可选多个取值）
6. 点击 **扫描计算**
7. 查看右侧曲线图和对应 3D 模型

> 曲线族取值支持 MATLAB 冒号表示法，如 `0.5:0.5:2.5`，表示乘以基准值的系数序列。

### 6.6 多规则体叠加

1. 切换到 **多规则体** 选项卡
2. 在"规则体列表"中点击 **+添加** 添加规则体，支持球体/圆柱体/长方体/椭球体/截锥体
3. 选中列表中某一规则体，在下方编辑其参数
4. 可修改规则体名称和颜色以区分
5. 点击 **计算** 进行叠加正演
6. 三维视图中显示所有规则体的叠加模型

## 7. 文件格式说明

### 7.1 导出数据（列式）

```
# 文本文件头部为注释行
# Component: Vz (muGal)
# X(m)  Y(m)  Vz(muGal)
0.000 0.000 1.234e+01
10.000 0.000 2.345e+01
...
```

每个分量一段，所有格点展平为一列。`.xlsx` 格式中每列对应一个分量。

### 7.2 ESRI ASCII Grid (.asc)

```
ncols         201
nrows         201
xllcorner     0.000000
yllcorner     0.000000
cellsize      10.000000
NODATA_value  -9999
1.234e+01 2.345e+01 3.456e+01 ...
...
```

标准 GIS 可交换格式，可被 ArcGIS / QGIS / GMT 等直接读取。NODATA 值固定为 -9999。

```

201 201
0.000000 2000.000000
0.000000 2000.000000
-12.345 56.789

...
```

Golden Software Surfer 的 DSAA（ASCII）格式，第 3-5 行依次为 X 范围、Y 范围、Z 范围。

```

0.000000 0.000000 1.234000e+01

...
```

每行一个格点，X / Y / Value 三列，以空格分隔。第一行为注释头。

### 7.6 观测面文件

支持从以下格式加载 2D 高程矩阵（行列结构与计算网格一致）：

- **TXT** — 空格分隔数值矩阵
- **DAT** — 同 TXT
- **ASC** — ESRI ASCII Grid 格式
- **GRD** — Surfer 6 DSAA 格式
- **XLSX** — Excel 工作表，第一张表（Sheet1）为数值矩阵

矩阵行数 = ny（Y 方向格点数），列数 = nx（X 方向格点数）。

## 8. 参数敏感性分析

敏感性分析面板允许用户系统地研究单个参数变化对正演结果的影响。

**工作机制：**
1. 选择一个参数作为 X 轴变量（从该规则体类型的可扫描参数中选择）
2. 设定 X 轴参数的 Min/Max 范围和步长数 N（均匀采样 N 个点）
3. 选择一个曲线族参数，以冒号表示法给出多个乘数因子（如 `0.5:0.5:2.5`）
4. 对每个 X 值，对每个曲线族乘数，计算所有格点的 Vz 峰值，绘制曲线族

**可扫描参数：** 各规则体以 "✓" 标记在 §3 的参数表中。

## 9. 多规则体叠加

多规则体功能支持任意数量、任意类型的规则体组合。

**叠加原理：** 总重力场 = 各规则体贡献的线性叠加（标量求和）：

$$V_z^{total}(x,y) = \sum_{k=1}^{N} V_z^{(k)}(x,y)$$

梯度分量同理。

**使用流程：**
1. 点击 **+ 添加** 添加规则体（默认命名为"球体1"、"球体2"……）
2. 在列表中选择规则体，编辑其类型、名称、颜色和参数
3. 重复添加所需的所有规则体
4. 点击 **计算** 执行叠加正演

**删除规则体：** 在列表中选中后点击 **删除**。
